# Devmaxx

**Maxx your DevEx.**

AI-powered business operations platform for Roblox game creators. Autonomous agents that maximize DevEx earnings — so creators focus on building great games, not running a business.

🌐 **[devmaxx.app](https://devmaxx.app)**

---

## What is Devmaxx?

Roblox creators are game designers, not business operators. They leave 20–40% of revenue on the table through unoptimized pricing, lose players from retention issues they never detect, and spend hours on support and analytics that should be automated.

Devmaxx connects to a creator's Roblox account via Open Cloud API and runs a suite of autonomous AI agents in the background — handling every non-creative task a game creator faces.

---

## The Agent Suite

| Agent | What it does | Runs |
|---|---|---|
| **MetricsMonitorAgent** | Pulls Open Cloud data daily, detects DAU drops and revenue anomalies | Daily |
| **PricingOptimizationAgent** | A/B tests Robux item prices, auto-implements winning prices | Weekly |
| **PlayerSupportAgent** | Classifies and resolves 85%+ of player messages automatically | Real-time |
| **CompetitorIntelligenceAgent** | Tracks top competing games, alerts on material changes | Daily |
| **ContentGenerationAgent** | Generates event ideas, item descriptions, pass concepts from real data | Weekly |
| **GrowthBriefAgent** | 90-second Sunday email: DevEx earnings, agent actions, top 3 next moves | Weekly |

## Tech Stack

- **Frontend:** Next.js 14, Tailwind CSS v3, Vercel
- **Backend:** Node.js, Railway, PostgreSQL (Prisma v5)
- **AI:** Anthropic Claude (claude-sonnet-4-20250514) — all agent reasoning
- **Orchestration:** n8n on Railway
- **Auth:** NextAuth v5 with magic link via Resend
- **Storage:** Cloudflare R2
- **Payments:** Stripe
- **Data:** Roblox Open Cloud API
